Cluster model: documents in the same topic folder are compared pairwise; cross-cluster pairs are never run. Adding 1 document to an existing cluster of size C triggers exactly C−1 new LLM calls — independent of the total corpus size N. This bounds ingestion cost at O(C), not O(N).

Full build = cost to analyze a fresh corpus from scratch. 1 addition = marginal cost of adding one document to an existing cluster (all other pairs cached).
